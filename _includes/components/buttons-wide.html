Wide Buttons

Use the class .button-wide when you need a button to fill the entire width of it's parent container.

Responsive Wide Buttons

Optional classes are available when you need a button to fill it's parent container only below a specific screen size. Add .button-wide-below-screen-mini, replacing -mini with -small, -medium, or -large. Try it out. Resize this browser window and observe when the buttons below switch to their wide view.
