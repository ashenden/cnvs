Containers provide basic layout functionality. The term "Container" has been borrowed from Twitter's Bootstrap, and is meant to imply an element that wraps or "contains" content. Properties such as a width and margin for the container vary depending on both the class and screen resolution. Choose from a responsive, fixed-width container (meaning its max-width changes at each breakpoint) or fluid-width (meaning itâ€™s 100% wide all the time).

Unlike Bootstrap, Containers in canvas can be used independent of the Grid System.

Fixed-Width Container

A container can be fixed or fluid. A fixed container is one where the width of the container is fixed value. That value may change based on the screen resolution to take advantage of additional horizontal screen space. Use the class .container to create a fixed with container. By default a fixed width container has no margin or padding. 

<div class="container">
  <!-- Container content goes here -->
</div>

Fluid-Width Container

Use .container-fluid for a fluid container. A fluid container is one where the width of the container is fluid relative to the width of it's parent. By default, a fluid container has outside margins on the left and right. The width of these margins can be modified with provided classes or removed entirely. For small devices, a fixed and a fluid container behave the same. 

<div class="container-fluid">
  <!-- Container content goes here -->
</div>

Fluid-Width Container Variants

Added classes have been provided to modify the outer margins applied to fluid-width containers. These can be used independently or in combination with other variant classes to created your desired layout arrangement.

Class Description
.container-fluid-flush Outside margins are removed entirely.
.container-fluid-narrow Outside margins by default are decreased by 50%
.container-fluid-narrower Outside margins by default are decreased by 25%
.container-fluid-wide Outside margins by default are increased to 200%
.container-fluid-wider Outside margins by default are increased to 400% 
  <div class="container-fluid container-fluid-narrow-right container-fluid-wide-left">
    <!-- Container content goes here -->
  </div>
