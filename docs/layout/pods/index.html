Pods are simple and convient way to adding space above and/or below your content. With provided classes, you can quickly define layout rules that fit the requirements of your project.

Use the class .pod to apply vertical padding to any component. By default, padding is applied evenly to both the top and bottom of the pod. However, with the provided classes you can override this to behave different — taller, shorter, or no padding at all.
