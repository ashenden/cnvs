Pod Options

By default, padding is applied evenly to both the top and bottom of the pod. You can easily adjust this behavior by adding a size modifier to the base .podclass. For example .pod-short will decrease the padding and .pod-tall will increase the padding. Furthermore, if you are seeking to change the padding in only a single direction, you can add an additional direction modifier. For example, .pod-tall-top increase the padding above the pod.

Size Modifiers

To adjust padding size apply one of the available pod-specific size classes. For example, simply using the class .pod-short will reduce padding evenly above and below the pod.

Class Description
.pod-flush Remove the padding in one or both direction entirely.
.pod-shorter Reduce the padding in one or both direction to 25%.
.pod-short Reduce the padding in one or both direction to 50%.
.pod-tall Increase the padding in one or both direction to 150%.
.pod-taller Increase the padding in one or both direction to 200%.

Size Direction Modifiers

Add -top or -bottom to the end of the size modifier class to adjust padding only in the direction implied by the direction modifier.

Class Description
.pod-*-top Adjust padding above the pod.
.pod-*-bottom Adjust padding below the pod.
