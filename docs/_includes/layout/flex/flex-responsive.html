Responsive Flex Box

The flex layout system in CNVS is built to be fully responsive across all included break-points. In your specific project it may be necessary to adjust flex behavior when moving from mobile devices to desktop screens. To apply as specific flex behavior at mini, small, medium, and large viewports, add the suffix -screen-mini, -screen-small, -screen-medium, or -screen-large respectively to your specific flex class.
