Modal Structure

To create a new modal use the .modal class. A modal is composed of 3 primary sections, all of which are optional and can be used interchangeably -- .modal-header, .modal-body, and .modal-footer. Looking for just the body and footer? It's as simple as not including .modal-header in your modal.

.modal-header
.modal-body
.modal-footer 
<div class="modal">
  <div class="modal-header">
    …
  </div>
  <div class="modal-body">
    …
  </div>
  <div class="modal-footer">
    …
  </div>
</div>

Modal Backdrop

Use the .modal-backdrop class to create an overlay that your modal lay on top of. This is optional, but is often nice to hide content behind the modal and keep your user's attention on the modal itself. The modal backdrop can also serve as a click-target for closing the modal.

Inverse Modal Backdrop

By default the modal backdrop element is dark, but can be easily inverted to a white appearance using .modal-backdrop-inverse.
